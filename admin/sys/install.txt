#установка

имя флешки
lsblk

размонтировать
sudo umount /dev/sdb1

форматнуть
sudo mkfs -t ext4 -L FLASH /dev/sdb1

создать установочную флешку
sudo dd bs=4M if=/home/r0_os/Downloads/debian-11.7.0-amd64-DVD-1.iso of=/dev/sdb status=progress oflag=sync


// инфа о вайфае
inxi -Nxxxz
//  Inxi сообщит, правильно ли ваш компьютер распознал ваш Wi-Fi-чип и загрузил правильные драйверы

sudo apt-get install grub
grub-install /dev/sda
update-grub

При загрузке установочного носителя BunsenLabs, когда появляется меню Grub, выберите желаемую запись (нажмите стрелку вверх или вниз) и нажмите клавишу TAB.
Press TAB at the menu entry an then add “ipv6.disable=1” just before "-- quiet", then the installation goes fine.
